# Heart Disease Prediction

This project predicts the presence of heart disease using machine learning.

## Tools & Libraries
- Python
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

## Algorithm Used
- Logistic Regression

## Project Workflow
1. Data loading
2. Data preprocessing
3. Train-test split
4. Model training
5. Model evaluation

## Results
- Training Accuracy: 85%
- Testing Accuracy: 81%

## How to Run
1. Clone the repository
2. Install required libraries
3. Open the notebook
4. Run all cells
